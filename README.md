# lga
currently working commands:
 - h,j,k,l: vim movements
 - g: go to first item
 - G: go to last item
 - dD: deletes a file or a directory, asks for confirmation when dir is not empty
 - backspace: toggle show hidden files
 - yy: yank
 - dd: move
 - p: paste
 - a: rename
 - `:touch`: touch file
 - `:mkdir`: mkdir dir
 - sn: sort by name
 - sm: sort by date modified
