# lga
currently working commands:
 - h,j,k,l: vim movements
 - g: go to first item
 - G: go to last item
 - dD: deletes a file or an empty dir
 - backspace: toggle show hidden files
 - yy: yank
 - dd: move
 - p: paste
 - a: rename
 - `:touch`: touch file
